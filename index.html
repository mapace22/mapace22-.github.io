Proyectos Destacados

A continuación, puedes explorar algunos de mis proyectos en los que he aplicado mis habilidades en el análisis y la ciencia de datos. Haz clic en cada uno para ver el código y la documentación completa en GitHub.

  • P1 - Análisis y Preparación de Datos para "Store 1": Limpieza, estandarización y corrección de datos de clientes, utilizando Python, métodos de cadenas y manejo de errores (try-except).
  • P2 - Análisis y Extracción de Datos de Clientes para "Store 1": Aplicación de funciones, bucles y condicionales en Python para segmentar clientes y calcular métricas de negocio.
  • P3 - Análisis de Preferencias Musicales: Springfield vs. Shelbyville: Manipulación de datos en un servicio de streaming para comparar hábitos de escucha y validar una hipótesis de negocio, utilizando Python y Pandas.
  • P4 - Limpieza y Análisis de Datos de Instacart: Limpieza, transformación y análisis de datos de compra para descubrir hábitos y fidelidad de clientes, utilizando Python, Pandas y visualizaciones.
  • P5 - Análisis Estadístico de Clientes en Megaline: Determinación de la rentabilidad de planes de telecomunicaciones mediante pruebas de hipótesis (t-Student) con SciPy y análisis de datos en Pandas.
  • P6 - Análisis de Patrones de Éxito en Videojuegos: Análisis de datos para identificar géneros y plataformas exitosas en el mercado, utilizando Python y pruebas de hipótesis para guiar una estrategia de marketing.
  • P7 - Aplicación Web Interactiva para Análisis de Datos de Autos: Creación y despliegue de una aplicación con Streamlit para el análisis exploratorio de datos de coches, utilizando Python, Plotly-Express y Render.com.
  • P8 - Análisis de Viajes en Taxi y su Relación con el Clima: Recopilación y análisis de datos de taxis en Chicago, usando SQL y web scraping en Python, para probar una hipótesis sobre el impacto del clima en el tiempo de viaje.
  • P9 - Análisis de Gastos de Marketing y ROMI para SHOWZ: Determinación del LTV y ROMI para optimizar la inversión en publicidad y mejorar la rentabilidad, utilizando Python y métricas de negocio.
  • P10 - Optimización de Ingresos y Análisis de Test A/B: Priorización de hipótesis con marcos ICE/RICE y análisis estadístico de un test A/B para optimizar la tasa de conversión, usando Python y SciPy.
  • P11 - Análisis de Comportamiento del Usuario y Test A/A/B: Análisis del embudo de ventas para identificar puntos críticos y evaluación de un cambio de diseño mediante un test A/A/B para optimizar la conversión, usando Python y SciPy.
  • P12 - Análisis de Tendencias de Vídeos en YouTube: Creación de un dashboard interactivo en Tableau y una presentación para analizar las categorías de vídeos más populares y el comportamiento de los usuarios. (Ver Dashboard)
  • P13 - Estrategia de Retención de Clientes con Machine Learning: Creación y evaluación de modelos predictivos (clasificación y clustering) para pronosticar la cancelación de clientes, utilizando Scikit-learn, Regresión Logística y Random Forest.
  • P14a - Análisis de Prueba A/B y Optimización de Conversión: Análisis de datos para evaluar un sistema de recomendaciones, validando la significancia estadística con la prueba Z y proporcionando conclusiones de negocio para optimizar el embudo de ventas.
